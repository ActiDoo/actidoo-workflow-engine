# Security Policy

## Reporting a Vulnerability

Please **do not** open public issues for security vulnerabilities.

Preferred reporting channel:
- Use **GitHub Security Advisories** for this repository ("Report a vulnerability").

If GitHub Security Advisories are not available for your account/org setup, contact the repository owners/maintainers via a private channel.

## Supported Versions

Security fixes are provided for:
- The **latest released version** (tags), and
- The default branch (`main`), until a release is cut.
